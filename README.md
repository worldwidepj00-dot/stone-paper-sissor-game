# Rock Paper Scissors Game

A fun, interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript. Play against the computer with real-time score tracking, round counter, and smooth animations.

## 🎮 Demo

Open `first.html` in any browser to play instantly.

## 🚀 Features

- Play Rock, Paper, Scissors against the computer
- Real-time score tracking for both player and computer
- Round counter to track total games played
- Win/loss/draw messages with color indicators
- Winner animation effect on each result
- Reset button to restart the game anytime
- Hover and click effects on choices for better user experience
- Responsive design with gradient background

## 🛠 Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling, animations, and responsive design
- **JavaScript (Vanilla)** – Game logic, DOM manipulation, event handling

## ▶️ How to Run

1. Download or clone this project
2. Make sure the `images` folder contains `rock.png`, `paper.png`, and `scissors.png`
3. Open `first.html` in any web browser
4. Click on any choice (Rock, Paper, or Scissors) to start playing

## 🎯 How to Play

- Click on Rock, Paper, or Scissors to make your choice
- The computer will randomly pick its choice
- Rock beats Scissors, Scissors beats Paper, Paper beats Rock
- Score updates automatically after each round
- Click **Reset Game** to start fresh
